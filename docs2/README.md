# Задание 3.1: REST API для регистрации пользователя

## Технические детали
* **Метод**: `POST`
* **URL**: `/api/v1/user/register`
* **Формат данных**: `application/json`

### Входные параметры
| Параметр | Тип | Обязательность | Ограничения |
| :--- | :--- | :--- | :--- |
| `firstName` | String | Да | Непустое поле |
| `lastName` | String | Да | Непустое поле |
| `userName` | String | Да | Уникальный идентификатор |
| `password` | String | Да | Мин. 8 символов, 1 цифра, 1 заглавная, 1 строчная, 1 спецсимвол |
| `recaptchaToken` | String | Да | Валидный токен Google reCAPTCHA |

### Коды ответов
| Код | Описание |
| :--- | :--- |
| `201` | Успешная регистрация пользователя |
| `400` | Ошибка валидации (пароль, обязательные поля или reCAPTCHA) |
| `409` | Конфликт (пользователь с таким `userName` уже существует) |
