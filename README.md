## Выявленные противоречия (Задача 1)

| № | Проблема | Комментарий |
|---|----------|-------------|
| 1 | Отсутствие таймаутов на ожидание поставщика | Нет ветки "поставщик не ответил" |
| 2 | Нет обработки отказа поставщика | Что если оборудование невозможно закупить? |
| 3 | Оценка сервиса не привязана к системе | Куда уходит оценка пользователя? |
| 4 | Дублирование систем №1 и №2 | Почему заявка создаётся в двух системах? |
| 5 | Логистика не имеет системы учёта | Только звонок, нет подтверждения получения |
| 6 | Нет проверки качества оборудования | Что если оборудование повреждено? |
